## Getting started

1. Install [Go](https://golang.org/doc/install)
2. Render a unique Genesis View.
3. Install the [mind](https://github.com/inconsiderable/focal-point/tree/main/mind)
4. Run the mind and issue a `newkey` command. Record the public key
5. Install the [client](https://github.com/inconsiderable/focal-point/tree/main/client)
6. Run the client using the public key from step 5. as the `-pubkey` argument

Complete steps for installation of Go and the ledger binaries on Linux can be found [here](https://gist.github.com/setanimals/f562ed7dd1c69af3fbe960c7b9502615).
